### Instructions

1. Using Python 3.12 run ```pip install -r requirements.txt``` to install all necessary packages. Or install them manually.
2. Run all cells in the Main.ipynb file and in the Main_with_noise.ipynb file.
3. You can now inspect the plots.

### Optional
1. To see the performance on a different datapoint, change the number variable in both files. Then run all cells again.
